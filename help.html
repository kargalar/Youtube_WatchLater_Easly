🎯 How to use

1. Add to Watch Later on Home

  1. Go to YouTube Home
  2. Right-click any video thumbnail
  3. The extension finds and clicks "Save to Watch Later"
  4. You’ll see a success notification

2. Remove from Playlist

  1. Open any playlist
  2. Right-click the video you want to remove
  3. The extension finds and clicks "Remove from playlist"
  4. The video is removed from the playlist

3. Supported pages

  • 🏠 Home — adds to Watch Later
  • 🔍 Search results — adds to Watch Later
  • 📺 Channel pages — adds to Watch Later
  • 📋 Playlist pages — removes from playlist
  • 🎬 Watch page — adds to Watch Later

🔧 Technical details

Multiple Method System

To improve reliability the extension tries multiple approaches:

  1. DOM Manipulation — finds buttons in the page
  2. Direct Button Search — searches for direct buttons
  3. Keyboard Shortcuts — uses key shortcuts
  4. Fallback — provides manual guidance

Smart playlist detection

How playlists are detected:

  • URL analysis (playlist parameters)
  • Page structure checks
  • Home page exclusions
  • DOM element presence

🚨 Troubleshooting

Common issues

  • Video not detected: Refresh the page
  • Right-click not working: Reinstall the extension
  • No notification: Set browser zoom to 100%
  • Wrong action: Check console for errors

Debug mode

If you’re having issues:

  1. Press F12
  2. Open the Console tab
  3. Right-click a video and observe console messages
  4. Share errors with the developer
